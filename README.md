# AI Resume Analyzer with Skill Gap Detection

## 🚀 Overview
This project is a Flask-based web application that analyzes resumes against job descriptions and provides a match score using NLP techniques.

## 🔧 Features
- Resume upload (PDF)
- Job description comparison
- Match score using cosine similarity
- Skill extraction and missing skill detection

## 🛠️ Tech Stack
- Python
- Flask
- scikit-learn
- PyPDF2
- HTML, CSS

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/Subharakshini/AI-Resume-Analyzer.git

2. Install dependencies:
   pip install flask PyPDF2 scikit-learn

3. Run the application:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000

## 📊 Output
- Match Score
- Extracted Skills
- Missing Skills
